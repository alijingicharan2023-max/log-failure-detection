# Predictive Configuration Failure Detection Using Logs

## 📌 Description
This project predicts system configuration failures using log analysis and machine learning.

## 🚀 Features
- Log analysis
- Failure prediction using ML
- Interactive UI using Streamlit
- Version control using Git

## 🧠 Technologies Used
- Python
- Scikit-learn
- Streamlit
- Git & GitHub

## ⚙️ How It Works
Logs → TF-IDF → Logistic Regression → Prediction

## ▶️ How to Run
1. Install dependencies:
   pip install pandas scikit-learn streamlit

2. Run the app:
   streamlit run app.py

## 📊 Output
- ⚠️ Failure Likely
- ✅ System Normal
